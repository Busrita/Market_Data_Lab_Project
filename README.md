Market Data Lab Project
Overview

This project analyzes the historical performance of three major U.S. equities:

Apple (AAPL)
Microsoft (MSFT)
Tesla (TSLA)

Using Python, pandas, NumPy, matplotlib, and yFinance, I conducted a market analysis focused on return, risk, diversification, and risk-adjusted performance.

Objectives

The project was designed to:

Retrieve real market data from Yahoo Finance
Compare stock performance using normalized returns
Calculate daily and cumulative returns
Measure risk through annualized volatility
Analyze diversification benefits using correlation
Evaluate risk-adjusted performance using the Sharpe Ratio
Produce an analyst-style investment recommendation
Tools Used
Python
Google Colab
yFinance
pandas
NumPy
matplotlib
Key Findings
Best Performing Stock

Tesla generated the highest cumulative return over the analysis period.

Highest Risk Stock

Tesla exhibited the highest annualized volatility, indicating significantly greater price fluctuations.

Best Risk-Adjusted Return

Apple produced the highest Sharpe Ratio, suggesting the strongest return per unit of risk.

Diversification Insights

Correlation analysis showed that Apple and Microsoft moved more closely together, while Tesla displayed lower correlation with both companies, providing diversification benefits.

Analyst Conclusion

While Tesla delivered the strongest absolute return, Apple demonstrated the most attractive risk-adjusted performance. A diversified portfolio containing all three securities would likely provide a better balance between growth potential and risk management than a concentrated position in a single stock.
